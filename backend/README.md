# ⚙️ 諸行無常ログ - Backend (API Server)

このディレクトリは、「諸行無常ログ」のバックエンド（サーバーサイド）のソースコードです。
主に **AIによる画像解析** や **定期的なバッチ処理（サボり判定）** を担当します。

## 🔧 技術スタック (Tech Stack)
* **言語**: Java 17+
* **フレームワーク**: Spring Boot 3.x
* **ビルドツール**: Gradle (または Maven)
* **データベース接続**: PostgreSQL (Supabase) / Firestore

## 🚀 役割 (What this server does)
アプリ本体（Flutter）では処理しきれない、以下の「重い処理」や「秘密の処理」を行います。

1.  **カロリー判定 API (`/analyze`)**
    * スマホから送られた食事画像を受け取る。
    * **Google Gemini API** に画像を投げて、カロリーと料理名を推定する。
    * 「2000kcalオーバー」などの判定結果を返す。
2.  **サボり判定バッチ (Scheduled Task)**
    * 決まった時間（例: 23:59）に起動。
    * 今日投稿していないユーザーをDBから探し出し、強制的に「劣化（ハゲ）」ステータスに更新する。
